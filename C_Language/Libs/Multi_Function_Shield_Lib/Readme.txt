Multi Function Shield Library - библитека для работы с одноименным шилдом
Язык - С
Среда разработки - AVR Studio 4.19 + AVR Toolchain 3.3.1.1020
Заточена под контроллер ATmega8 (системный таймер сделан на базе прерывания по совпадению Timer1)
Cостав:
1. Multi_Function_Shield_Lib.c - файл библиотеки
2. Multi_Function_Shield_Lib.h - файл заголовка
3. Main.c - файл примера использования функций

Возможности:
1. Выбор одного из 3 режимов работы с помощью опроса 3-х кнопок (в главном цикле, либо по таймеру). Выбор сопрово
2. Режим 1 - Опрос 0 канала АЦП (потенциометр) с уходом в сон и пробуждением по окончанию преобразования с вызовом прерывания.
3. Режим 2 - Опрос инфракрасного приемника (TSOP) в протоколе RC5 (с помощью внешнего прервания INT0, либо поллингом. В зависимости от этого меняется способ опроса кнопок. Способ опроса инфракрасного приемника задется при компиляции исходника).
4. Режим 3 - Опрос показаний температуры датчика DS18B20 (только 2 первых байта, без контроля CRC). В случае отсутствия датчика - вывод на индикатор сообщения "Err"
5. В зависимости от режима, выбраного кнопками - вывод показаний АЦП, инфракрасного приемника либо температурного датчика на 7-сегментный идикатор через 2 последовательно соединенных 74НС595.
